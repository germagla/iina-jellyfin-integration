Jellyfin Library

Connect, browse, and manage your Jellyfin account in the standalone library window. You can also open it from IINA’s Plugins menu or press ⌥⌘J from any focused IINA window.

Playback

Enter exact chapter titles separated by commas. Matching chapters automatically advance to the next chapter.

Diagnostics

Jellyfin for IINA keeps a small rotating, redacted local log to make playback problems easier to investigate. Authentication secrets and media URLs are never recorded.

Connection and Privacy

Server and account changes are made in the Jellyfin Library. Access tokens are stored in macOS Keychain and are never exposed to this preferences page.
